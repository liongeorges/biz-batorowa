## 技術

- **バニラ HTML / CSS / JavaScript のみ**（フレームワーク・ビルドツール・バックエンドなし）
- 静的ホスティング（GitHub Pages）／ **localStorage 不使用**
- 診断本体1ファイル ＋ レポート8枚 ＋ 8ジョブ図鑑
- 計測：GA4 ／ リード回収：LINE公式アカウント（キーワード完全一致） ／ 拡散：X intent

## 設計上の特徴

- **タイブレークの順序を全40320順列で総当たり探索**し、結果分布の格差を **8.1倍 → 3.19倍** に圧縮
- **色・ラベル・設定値をすべて単一情報源に集約**（`theme.css` の8行 ／ `PARAM_LABELS` ／ `CONFIG`）。後から差し替え可能な構造
- 診断エンジンは**ドメイン非依存**。アーキタイプ集合を差し替えれば別テーマに再スキンできる

## 構成

```
index.html          診断本体
theme.css           ★ジョブのアクセント色（唯一の定義場所）
report/             レポート8枚 ＋ 8ジョブ図鑑
spec/               仕様書（実装の正典）
scripts/            スコアロジックの検算スクリプト
```

検算：

```bash
node scripts/verify_scoring.mjs
```
